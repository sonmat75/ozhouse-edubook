# ozhouse-edubook 프로젝트

## 프로젝트 개요
오즈하우스의 환경 주제 활동책 시리즈 개발 프로젝트.
미취학 6~7세 아이와 부모가 함께 하는 활동책을 에이전트 파이프라인으로 제작한다.

## 오즈하우스 소개
- 15년 역사의 환경교육 전문 사회적기업
- 보드게임, 교구, 교재 등 다양한 환경교육 콘텐츠 개발
- 브랜드 철학: 저절로 교육, 재미 우선, 포용적 설계
- 환경캐릭터 '푸리' 보유 — 이번 시리즈에 활용

## 시리즈 기본 설정
- 대상: 미취학 6~7세
- 형태: 부모와 아이가 함께 하는 활동책 (전체 기본값)
- 페이지 수: 48페이지 내외 (표지 포함, 조정 가능)
- 캐릭터: 오즈하우스 환경캐릭터 '푸리' 활용
- 첫 권 주제: 재활용 / 분리배출

## 에이전트 구성 및 파이프라인
오케스트레이션 없이 미노(담당자)가 직접 단계별로 호출하고 검수한다.

```
planner.md          → ① 기획 에이전트
activity_designer.md → ② 활동 설계 에이전트
text_writer.md      → ③ 텍스트 에이전트
illust_prompter.md  → ④ 일러스트 프롬프트 에이전트
layout_editor.md    → ⑤ 편집 레이아웃 에이전트
```

### 단계별 흐름
```
① 기획 (단원 구성 + 학습목표)
        ↓ 미노 검수·승인
② 활동 설계 (활동 유형 + 페이지 구조)
        ↓ 미노 검수·승인
③ 텍스트 (지시문 + 설명문)
        ↓ 미노 검수·승인
④ 일러스트 프롬프트 (그림 단위 영문 프롬프트)
        ↓ 미노 검수·승인
⑤ 편집 레이아웃 (페이지 배치 지시서 → PPTX 초안 → Figma 이관)
```

### 최종 결과물 흐름
```
에이전트 PPTX 초안 → 미노 검토·수정 → Figma 디자인 완성 → 인쇄용 PDF
```

## 작업 원칙
- 모든 단계는 미노가 직접 검수한 후 다음 단계로 넘어간다
- 에이전트는 각 단계의 초안 생성자 역할이며 확정권은 미노에게 있다
- 추후 필요 시 검수 에이전트 추가 가능
- 모든 결과물은 한국어로 작성
